
# Linux Journey 🐧

Hey, I'm Ananthakrishna. I'm documenting my Linux learning path here — one day at a
time. Nothing fancy, just honest notes on what I learned, what confused me, and
what actually made sense after a while.

I'm using KodeKloud labs alongside this, so most of what's here has been
actually typed out in a real terminal, not just read about.

---

## Why I'm doing this

I want to get solid at Linux and shell scripting from the ground up. Not just
enough to get by, but actually understand what's happening when I run a command.
This repo is how I hold myself accountable.


## How this repo is organized

Each day has its own folder with a README inside. That README has what I
learned, commands I practiced, things that tripped me up, and what I'm doing
next.
